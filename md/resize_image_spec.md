# spec: Resize IMAGE

How it works on iLoveIMG & how we implement a custom premium Dribbble UI for it.

---

## 1. iLoveIMG Features
- **Resize by Pixels:** Specific width and height with optional lock aspect ratio.
- **Resize by Percentage:** Fast scaling by 25%, 50%, or 75%.
- **Safety options:** "Do not enlarge if smaller" checkbox.

---

## 2. Our Custom Premium UI (Dribbble-Inspired)
- **Dual Tab Interface:** Toggle smoothly between "By Pixels" and "By Percentage".
- **Percentage Quick Selector:** Beautiful pill-shaped buttons for 25%, 50%, and 75% with hover/active animations.
- **Aspect Ratio Locking Indicator:** Chain link icon between Width and Height inputs that visually links them together.
- **Safety Toggle:** Styled check button for "Do not enlarge if smaller".
