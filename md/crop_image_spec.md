# spec: Crop IMAGE

How it works on iLoveIMG & how we implement a custom premium Dribbble UI for it.

---

## 1. iLoveIMG Features
- **Custom cropping coordinates:** Set width, height, position X, and position Y.
- **Visual overlay bounding box:** Users adjust borders directly.

---

## 2. Our Custom Premium UI (Dribbble-Inspired)
- **Preset Ratios Panel:** A section containing fast aspect ratio presets:
  - **Custom (Freeform)**
  - **1:1 Square (Profile)**
  - **16:9 Landscape (Video)**
  - **9:16 Portrait (Stories/Reels)**
  - **4:3 Standard**
- **Unified Coord Form:** Stacked clean input fields for Width, Height, X, and Y with right-aligned value readouts.
