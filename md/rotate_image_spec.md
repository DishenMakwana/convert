# spec: Rotate IMAGE

How it works on iLoveIMG & how we implement a custom premium Dribbble UI for it.

---

## 1. iLoveIMG Features
- **Clockwise rotation:** Rotate by 90°, 180°, or 270°.
- **Orientation filters:** Portrait vs Landscape bulk handling.

---

## 2. Our Custom Premium UI (Dribbble-Inspired)
- **Smart Rotation Dial:** Interactive layout depicting clockwise/counter-clockwise rotations.
- **Degree Badges:** Visual angle indicators showing 90°, 180°, and 270°.
- **Filter Settings:** Let the user select target orientation constraints (e.g. only rotate landscape, only rotate portrait).
